# About
The ATM Banking Management System is a Java project that works like a real ATM. It allows users to create an account, log in, check their balance, deposit money, withdraw money, change their PIN, and view transaction history.

# Features
- User Login & Signup
- Balance Enquiry
- Deposit Money
- Withdraw Money
- PIN Change
- Mini Statement
- Transaction History

# Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

# How to Run
1. Clone or download the project.
2. Open it in Eclipse.
3. Import the MySQL database.
4. Update database details in `Conn.java`.
5. Run `Login.java`.
